# Capsule Mobile Agent Guide

This is the native Expo app for iOS and Android.

- Use Expo SDK 56 docs for Expo-specific implementation details.
- Target development builds/custom dev clients. Do not build the browser web experience here.
- Keep mobile code in this directory and shared API contract notes in `../../docs/api`.
- Browser invite/contribution flows belong to the Laravel Inertia React app in `../backend`.
